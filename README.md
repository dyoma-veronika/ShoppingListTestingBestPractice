# ShoppingList
Shopping list application with Pixabay API

Covered with tests:
- Adding of items
- Item deletion with swipe
- Image picking
- Navigation
- Fragments
- Room DB
- View Models

Stack:
Room, Retrofit2, Hilt, Espresso, JUnit4, Mockito, Truth
